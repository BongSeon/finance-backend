## 사용 방법

### 서버 실행

3000번 포트가 실행됩니다.

```
npm start
```

> ### API
>
> #### 조회 : GET
>
> ```
> http://localhost/api/consults
> ```
>
> #### 추가 : POST
>
> ```
> http://localhost/api/consult/insert
> ```
>
> #### 수정 : PUT
>
> ```
> http://localhost/api/consult/update
> ```
>
> #### 삭제 : DELETE
>
> ```
> http://localhost/api/consult/delete/:id
> ```
